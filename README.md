## Symfony4 REST Api boilerplate
Author: **dmk-web**

**Project init**
`make init`

**Base URL**: http://localhost:8080 or http://project.dev

# Endpoints
### Auth:
* GET /token - **get token by login and pass**

### User:
* GET /users - **get list of users**
* GET /users/{uuid} - **get user by id**
* POST /users - **create new user**
